## What Went Well

- **Single damage authority preserved:** Planet impact decrements HP, resets combo, and feeds the same loop that Epic 2 guarded; no parallel “mystery damage” paths showed up in the artifacts.
- **Pure logic + tests scaled:** `atmospheric-health`, `combo-engine`, `gold-fragment`, and `intensity-shatter-fsm` each got focused unit tests; review follow-ups tightened edge cases (combo cap emits, NaN resistance, spectacle intensity parity).
- **Shader stack extended without abandoning performance rules:** Health tint composes in the existing heartbeat filter; intensity drives oscillation and collision spectacle multipliers from config.
- **Clear integration seams for Epic 4:** `PLANET_SHATTERED`, `INTENSITY_STAGE_CHANGED`, and frozen storm behavior on shatter give the next epic explicit hooks.
- **Gold debris composed with existing systems:** Spawn roll, pooled probes, impulse scaling, metallic burst, and `SCORE_AWARDED` payload extension stayed inside established pooling and hit-resolution patterns.
- **Post-review hardening culture:** Several stories document explicit CR passes (filter placement, tier-change-only combo events, spawner ordering vs FSM sync).

---

## Challenges and Patterns

- **Bootstrap orchestration load (recurring theme):** Epic 2 called this out; Epic 3 confirmed it — ordering notes (e.g. FSM sync before spawner update, planet-impact vs flick tick order) are critical and easy to regress.
- **Visual/shader footguns:** Filter target hierarchy (body vs root), padding, and alpha-aware rim math required iteration; uniform and packing debt from Epic 2 remains a live concern when adding more full-screen or overlay effects.
- **Documentation vs implementation drift:** Planning `epic-list.md` references 4.1a/4.1b split; sprint artifacts use consolidated `4-1-results-overlay-one-more-try` — worth reconciling before deep Epic 4 dev.
- **Epic 2 retro action items — mixed follow-through:** Baseline note, uniform checklist, bootstrap façade, damage-pipeline doc, QA checklist template, and time-scale integration harness were not uniformly evidenced as completed *artifacts* in Epic 3 story records (see next section). The *technical outcomes* still largely reflect those intents (single pipeline, tests, manual smoke).
